2강 data type
data type은 bool, bytes, address, uint 타입이 존재한다.
bool type은 bool public b = false; 처럼 쓸 수 있다.
다른 언어처럼 비트 연산자 논리연산자도 존재한다.
address란 계좌를 통해서 돈을 주고 받는 계좌번호 같은 존재이다. ethereum을 송수신 할 때 사용한다.
address는 사람한테만 부여되는 게 아니라 smart contract를 deploy할 때도 address가 생긴다.
int8은 8비트. -2^7 ~ 2^7-1 int uint8은 0 ~ 2^8-1. 보통은 uint를 많이 사용한다.
